# Python-Apply-Capture

# 概要

ウィンドウ名の一部を指定して、最初に取得できたウィンドウを、定期的にpngで保存します。
(描画領域のみではなく、タイトルバーなどを含みます)

ウィンドウ名と取得間隔を指定できます。

# 動作検証環境

* Windows7 64bit
* Python 2.7
